# How to Compile

## compile.sh

Run `./compile.sh filename.xc` to generate the executable `filename`.

This script requires:
* `../artifact/ableC.jar`
* `../include/sqlite.xh`
* either:
  * installation of the SQLite shared library and header files; or
  * `../sqlite/sqlite3.c`
